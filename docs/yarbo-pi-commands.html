Yarbo Control Panel — Raspberry Pi Quick Reference

Pi: admin@192.168.9.253  |  Panel URL: http://192.168.9.253:8080

SSH into the Pi

ssh admin@192.168.9.253

Edit configuration

nano ~/yarbo/config.php

Common settings: broker_host, serial, cameras_enabled

  • Save: Ctrl+O, then Enter
  • Exit: Ctrl+X

Restart the service (after saving config)

sudo systemctl restart yarbo-panel

Check the service is running

sudo systemctl status yarbo-panel

Press q to exit.

View recent logs

sudo journalctl -u yarbo-panel -n 30

Follow logs live

sudo journalctl -u yarbo-panel -f

Press Ctrl+C to stop.

Test API from the Pi

curl -s http://127.0.0.1:8080/api/status.php

Test connection to Yarbo MQTT

nc -zv 192.168.1.223 1883

Replace 192.168.1.223 with your Yarbo IP if it has changed.

Edit the systemd service (rare)

sudo nano /etc/systemd/system/yarbo-panel.service

After changing the service file:

sudo systemctl daemon-reload
sudo systemctl restart yarbo-panel

Handy commands

TaskCommand
Stop servicesudo systemctl stop yarbo-panel
Start servicesudo systemctl start yarbo-panel
Disable auto-start on bootsudo systemctl disable yarbo-panel
Enable auto-start on bootsudo systemctl enable yarbo-panel
Reboot Pisudo reboot
