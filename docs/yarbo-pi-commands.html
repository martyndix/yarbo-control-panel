Yarbo Control Panel — Raspberry Pi Quick Reference

Panel URL: http://<pi-ip>:8080  |  Install: sudo ./scripts/install.sh --deps

First-time install

git clone https://github.com/martyndix/yarbo-control-panel.git ~/yarbo
cd ~/yarbo && sudo ./scripts/install.sh --deps

Open the panel URL in a browser → Settings → enter Yarbo broker IP and serial.

SSH into the Pi

ssh <user>@<pi-ip>

Change broker IP or serial (recommended)

Use the web Settings page — no file editing required.

Advanced: edit ~/yarbo/config.php only if needed.

Restart the service

sudo systemctl restart yarbo-panel

Check the service is running

sudo systemctl status yarbo-panel

Press q to exit.

View recent logs

sudo journalctl -u yarbo-panel -n 30

Follow logs live

sudo journalctl -u yarbo-panel -f

Press Ctrl+C to stop.

Test API from the Pi

curl -s http://127.0.0.1:8080/api/status.php

Test connection to Yarbo MQTT

nc -zv <yarbo-ip> 1883

Update panel (after git clone install)

cd ~/yarbo && ./scripts/update.sh

Or open the panel → Settings → Panel updates → Update to latest.

Re-run installer (update paths / service)

cd ~/yarbo && sudo ./scripts/install.sh

Handy commands

TaskCommand
Stop servicesudo systemctl stop yarbo-panel
Start servicesudo systemctl start yarbo-panel
Disable auto-start on bootsudo systemctl disable yarbo-panel
Enable auto-start on bootsudo systemctl enable yarbo-panel
Reboot Pisudo reboot
